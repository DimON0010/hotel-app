## Description

Hotel-app backend 

Use the commands from the root folder
## Installation

```bash
$ npm install
```

after that 
```
cp local.env .env
```
## Starting the database

```bash
$ npm run docker-compose
```
## Initialize database

```bash
$ npm run initialize-db
```

## Seed the database

```bash
$ npm run seed
```
## Running the app

```bash
$ npm run start:dev
```

## Test

```bash
# unit tests
$ npm run test
```
